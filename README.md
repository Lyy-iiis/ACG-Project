## Installation

1. Clone the repository
```bash
git clone git@github.com:Lyy-iiis/ACG-Project.git
```

2. Install the required packages
```bash
conda env create -f environment.yml
```

3. Activate the environment
```bash
conda activate acg
```

4. Run the main.py
```bash
python main.py
```
